```text
# Mars Explorer

Веб-приложение на Flask для управления миссией колонизации Марса.

---

## Возможности проекта

- Регистрация и авторизация пользователей
- Профили космонавтов
- Загрузка аватарок
- CRUD-система для работ
- CRUD-система для департаментов
- Категории работ
- REST API
- Карта родного города пользователя
- Обработка ошибок 400 и 404
- Bootstrap-интерфейс

---

## Используемые технологии

- Python
- Flask
- Flask-Login
- Flask-WTF
- Flask-RESTful
- SQLAlchemy
- SQLite
- Bootstrap 5

---

## Структура проекта

- `main.py` — основной файл приложения
- `data/` — модели базы данных и API
- `templates/` — HTML-шаблоны
- `static/` — CSS и изображения
- `db/` — база данных SQLite

---

## Установка и запуск

### 1. Установите зависимости

pip install -r requirements.txt

### 2. Запустите приложение

python main.py

После запуска приложение будет доступно по адресу:

http://127.0.0.1:8080/
```
